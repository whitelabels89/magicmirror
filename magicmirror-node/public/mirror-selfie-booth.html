📲Pengiriman WhatsApp

1. Render & Upload Generate gambar + watermark → upload ke mini PC storage atau Firebase Storage.
2. Kirim Lewat Whacenter API Gunakan endpoint existing dengan payload nomor customer & URL gambar. { device: WHACENTER_DEVICE, to: "+62xxxx", mediaUrl: "https://.../selfie.jpg" }
3. Pesan Otomatis "Hai [Nama Customer], ini hasil Mirror Selfie kamu di Queen's Salon 💖. Terima kasih sudah berkunjung!" + CTA promo.
Smart Contact Simpan nomor ke profil customer di Google Sheets → mirror tinggal memilih kontak saat share.
Security Berikan persetujuan di awal treatment: customer setuju menerima foto via WhatsApp.
Tracking Log waktu pengiriman dan status delivered/read untuk analitik kampanye.
