Hai, Pembuat Game Hebat!

Hari ini kita akan berbagi game seru yang sudah kamu buat dengan keluarga dan teman!

Siapkan Game Kamu 🕹

Pastikan game yang kamu buat sudah berjalan lancar.

Ceritakan Game Kamu 🎤

Jelaskan cara bermain, karakter, dan rintangan seru di game buatanmu.

Tunjukkan Game Kamu 📱

Perlihatkan game ke keluarga atau teman dengan percaya diri.

Dapatkan Apresiasi 👏

Terima pujian karena sudah berani membagikan game buatanmu.

Hebat!

Hari ini kamu belajar berbagi karya game dengan orang lain. Siap buat game lebih keren di project berikutnya?

Quiz Part 1 – Teori

  1. Kenapa penting berbagi game yang kita buat?
  2. Apa yang bisa kamu ceritakan saat membagikan game?
  3. Sebutkan dua hal yang perlu disiapkan sebelum presentasi game.
  4. Bagaimana cara menerima apresiasi dari orang lain?
  5. Apa yang bisa kamu katakan untuk memuji game temanmu?

Quiz Part 2 – Praktik

Tugas:

  1. Pastikan game kamu sudah berjalan lancar.
  2. Ceritakan game kamu ke orang tua atau teman dengan percaya diri.
  3. Jelaskan cara bermain dan keseruan game buatanmu.
  4. Foto momen saat kamu membagikan game.
  5. Tunjukkan foto ke guru atau orang tua untuk mendapat apresiasi!
