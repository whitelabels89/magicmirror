Apa itu Pernyataan Kondisional Bersarang?

Pernyataan bersarang terjadi ketika kita menuliskan if di dalam blok if lainnya. Cara ini membantu kita memeriksa kondisi yang lebih dari satu tingkat.

Bayangkan kamu sedang memilih menu makan siang:

  • Jika kamu lapar, cek apakah kamu ingin makanan berat atau ringan.
  • Jika makanan berat, pilih nasi atau mie.
  • Jika makanan ringan, pilih roti atau buah.

Ini sama seperti nested conditional, di mana keputusan di dalam keputusan lain.

if lapar:
    if ingin_makanan_berat:
        pilih nasi atau mie
    else:
        pilih roti atau buah

Kenapa Pakai Nested, Bukan Kondisi Panjang?

Dengan menulis kondisi di dalam kondisi, kode menjadi lebih rapi dan mudah dibaca daripada menulis kondisi yang panjang pada satu baris.

Selain itu, nested conditional membantu mengelompokkan logika yang terkait, sehingga lebih mudah dipahami dan dikelola.

Contoh:

if usia >= 18:
    if punya_ktp:
        boleh_mengemudi = True
    else:
        boleh_mengemudi = False
else:
    boleh_mengemudi = False

Aturan Menulis Nested Conditional

  • Gunakan indentasi 4 spasi untuk setiap tingkat.
  • Pastikan setiap if diakhiri dengan tanda titik dua :.
  • Tulis kondisi dari yang paling umum ke yang lebih spesifik.
  • Jangan lupa untuk menuliskan else atau elif jika diperlukan agar semua kemungkinan tertangani.

Contoh:

if cuaca == 'hujan':
    if bawa_payung:
        keluar = True
    else:
        keluar = False
else:
    keluar = True

Kesimpulan Nested

Nested conditional memungkinkan kita untuk membuat keputusan yang lebih kompleks dengan mengecek kondisi bertingkat secara berurutan.

Pastikan indentasi benar agar program berjalan sesuai harapan dan mudah dibaca.

Gunakan nested conditional untuk mengelompokkan logika yang saling terkait dan memudahkan debugging.

Kondisional dengan Banyak Cabang

Kita bisa menggunakan elif untuk menambah cabang keputusan selain if dan else.

Ini berguna saat ada banyak kondisi yang harus diperiksa secara berurutan.

Contoh:

if nilai >= 90:
    print('A')
elif nilai >= 80:
    print('B')
elif nilai >= 70:
    print('C')
else:
    print('D')

Dengan elif, kode menjadi lebih bersih dan mudah dibaca daripada menulis nested if bertingkat.

Kenapa Pakai elif dan else?

elif dan else membantu menangani berbagai kemungkinan tanpa harus menulis banyak if terpisah.

Ini membuat kode lebih efisien dan mudah dipahami.

Selain itu, penggunaan elif mencegah pengecekan kondisi yang tidak perlu setelah kondisi sebelumnya terpenuhi.

Kesimpulan Akhir

Dengan nested conditional dan elif, kita dapat membuat program yang menangani banyak situasi berbeda secara terstruktur.

Penggunaan struktur ini membantu membuat kode lebih bersih, mudah dibaca, dan mudah dipelihara.

Selalu perhatikan indentasi dan logika agar program berjalan sesuai yang diharapkan.
