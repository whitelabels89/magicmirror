Quiz Part 1 – Teori

  1. Apa gunanya atribut warna pada object?
  2. Bagaimana warna mempengaruhi aksi object?
  3. Apa itu kombinasi atribut & method?
  4. Contoh kode class dengan atribut & method aksi tergantung warna.
  5. Cara buat dua object dengan warna berbeda.
  6. Apa hasil kode:
    class Bola:
    def __init__(self, warna):
        self.warna = warna
    def pantul(self):
        if self.warna == 'kuning':
            print('Bola kuning memantul tinggi!')
        else:
            print('Bola', self.warna, 'memantul biasa.')

b = Bola('kuning')
b.pantul()

Quiz Part 2 – Praktik

Tugas:

  1. Buat class dengan atribut nama & warna, method mencetak keduanya.
  2. Tambah method aksi berdasarkan warna (misal: merah=ngebut, biru=santai).
  3. Buat 2 object dengan warna berbeda & panggil semua method.
  4. Buat class lain dengan atribut & method yang berubah sesuai warna.
  5. Modifikasi class agar method juga mencetak warna saat dipanggil.
  6. Buat class dengan atribut & method mengubah gaya bicara sesuai warna.
  7. Buat object hitam & putih, panggil method keduanya.
  8. Buat class dengan method cetak gaya cahaya sesuai warna.
  9. Tambah fitur: object menyebutkan dirinya → "Saya [class] berwarna [warna]".
  10. Desain class bebas (peri, robot, kendaraan, buah) dengan atribut warna & method unik + komentar.
