Langkah 1 – Rancang Proyekmu

Bayangkan kamu bikin game petualangan:

  • Modul berisi fungsi lari(), serang(), upgrade()
  • File utama memanggil jurus dari modul
  • Gambar dulu rancangannya di kertas: nama modul, fungsi, dan interaksi antar file
