Mengapa Kita Belajar Python?

Python mudah dipelajari dan banyak dipakai di industri.

Apa yang Bisa Dilakukan Python?

  • Web development
  • Data science
  • Automasi tugas
  • Kecerdasan buatan

Pentingnya Dasar Teori Developer

Memahami teori memudahkan kita menulis kode yang rapi dan efisien.

Konsep Dasar yang Harus Dipahami

  • Algoritma
  • Bahasa Pemrograman
  • Program

Apa Itu Algoritma?

Serangkaian langkah logis untuk menyelesaikan masalah.

Karakteristik Algoritma yang Baik

  • Jelas dan terurut
  • Tidak menimbulkan keraguan
  • Efisien dijalankan

Contoh Algoritma

Resep membuat kue: siapkan bahan, campur, panggang, sajikan.

Kode yang Bisa Dieksekusi

Coba jalankan kode berikut.

// Hasil akan muncul di sini

Apa Itu Bahasa Pemrograman?

Bahasa khusus untuk memberi instruksi kepada komputer, salah satunya Python.

Apa Itu Program?

Kumpulan algoritma yang ditulis dengan bahasa pemrograman.

Cara Menulis Program yang Bisa Dipahami Komputer

Tulis sintaks dengan benar dan ikuti urutan logis.

Kesimpulan Materi Dasar Python

Python populer dan mudah dipelajari, pahami konsep dasar sebelum praktik.

Task di Platform

Kerjakan latihan pengantar Python pada platform iCoding.

Apa Itu Fungsi?

Fungsi mengelompokkan perintah yang bisa dipanggil berulang.

Fungsi print()

Digunakan untuk menampilkan teks atau angka ke layar.

Mencetak Nilai dengan print()

// Hasil akan muncul di sini

print() dengan Ekspresi

// Hasil akan muncul di sini

Sintaks Fungsi dan Operasi Lain

Penulisan fungsi memakai tanda kurung, operasi mengikuti aturan matematika.

Urutan Operasi dalam Ekspresi

Perkalian dan pembagian dikerjakan sebelum penjumlahan. Gunakan kurung bila perlu.

Kesimpulan Fungsi

Fungsi membantu menata kode. print() adalah contoh paling dasar.

Task di Platform
